# Game Values web UI widgets library

## Setup

Install the dependencies:

```bash
# yarn
yarn install
```

## Development

Start the development server on http://localhost:3000

```bash
# yarn
yarn dev
```

Then browse to http://localhost:3000 to see the homepage.
